Internet

A global computer network providing a variety of information and communication facilities, consisting of interconnected networks using standardized communication protocols.

Web

A collection of public web pages that are connected by hyperlinks and accessible through the internet.

How are the Internet and Web different?

The internet is a global network of computers that connects to other networks, while the web is a service that runs on the internet

ASCII and Unicode

ASCII is an older standard that supports a limited number of characters, while Unicode is a more recent standard that supports many more characters and symbols.

What is UTF-8

UTF-8 (Unicode Transformation Format 8) is a character encoding system that represents characters using one to four bytes.

URL

A URL (Uniform Resource Locator) is a text string that specifies the location of a resource on the internet.

Domain Name

A unique name that identifies a website, email service, or other resource on the internet. It's a string of text that users type into a browser to access a website.

HTTP

"HTTP" in computer terms stands for "Hypertext Transfer Protocol," which is the protocol used to transfer information on the World Wide Web

HTTPS

HTTPS stands for Hypertext Transfer Protocol Secure, and it's a secure version of HTTP. HTTPS is used to encrypt data that's transferred between a web browser and a website.

IP Address

An IP address, or Internet Protocol address, is a unique number that identifies a device on the internet. IP addresses are used to send and receive data between devices, and are essential for the internet to work

DNS

DNS stands for Domain Name System, which is a system that translates domain names into IP addresses. DNS is a critical part of the internet that allows users to access websites by typing in human-readable domain names.

W3C Recommendations

A W3C Recommendation is a set of guidelines or technical specifications that the World Wide Web Consortium (W3C) endorses.

XML, XHTML, HTML, HTML5

In the context of computers and the web, XML, XHTML, HTML, and HTML5 are all markup languages used to create and structure web pages and documents. They define how content is displayed and organized on the web
