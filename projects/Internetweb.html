Internet

It is a set of computer networks that offers us different informations.

Web

It is a set of web pages that can be accessed through the internet.

How are the Internet and Web different?

The internet connects computers around the world and the web is a kind of service that runs on the internet

ASCII and Unicode

ASCII is only for English text while Unicode supports different characters from different languages.

What is UTF-8

UTF-8 (Unicode Transformation Format 8) is used for representing characters like numbers, letters, and different symbols

URL

A URL (Uniform Resource Locator) is used to locate specific resources on the internet.

Domain Name

It is a name that is used for the address of a website.

HTTP

"HTTP" in computer terms stands for "Hypertext Transfer Protocol". It is used to load web pages.

HTTPS

HTTPS stands for Hypertext Transfer Protocol Secure, and it is a version of HTTP that is more secure. It is encrypted so that the security of data transfer would increase.

IP Address

An Internet Protocol address is a set of numbers that allows the communication of other devices over the internet

DNS

DNS stands for Domain Name System. It turns the name of a domain into an IP address.

W3C Recommendations

A W3C Recommendation enhances the privacy of a user and secures their communication. It is a set of guidelines that the World Wide Web Consortium (W3C) endorses.

XML, XHTML, HTML, HTML5

XML (eXtensible Markup Language) shows how the data is transferred. It is basically designed for storing and transporting data
XHTML (eXtensible Hypertext Markup Language) is another version of HTML that was developed to improve the reliability of web pages
HTML (Hypertext Markup Language) is the standard language that we use for creating different web pages.
HTML5 is the latest version of HTML that supports newer applications on the web
