What is the difference between OL and UL Tags?

OL or ordered list starts with the "OL tag and each item starts with the "li" tag. The list items will be marked with numbers like this:

  1. Walmart
  2. best Buy
  3. Foodland
UL or unordered list starts with the "ul" tag and each item list has to start with the "li" tag. The list items will be marked with bullets (small black circles) like this:
  • Target
  • Walmart
  • Seafood City
