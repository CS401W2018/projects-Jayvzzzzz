Questions:

Explain the difference between inline, embedded, and external CSS.

Inline CSS

For specific elements only

Embedded CSS

For a specific document but only applied globally in that document

External CSS

Separate stylesheet that can be used on multiple pages

Explain why you should only use external CSS.

- It makes your code cleaner and it also gives way for the website editors to freely adjust the visuals of the website

What is a selector used for?

- It is used to change specific elements within an HTML that you want to style
