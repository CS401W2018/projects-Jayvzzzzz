What is the difference between block tags and inline tags?

Block tags has width and height that can be set while inline tags don't have that because they are surround text.

What is the margin, border, and padding?

Margin - outside the block

Padding - inside the block; pushes it inside

Border - the border of the block

What are units of measurement you can use in CSS?

Medium, inches, centimeters, thin (fixed size measurements)

Relative sized measurements - %, em

When would you use a figure tag?

For inserting an image without messing up the flow of the webpage
