List out the different ways to select an item.

  • Tag Name
  • Descendent Rule
  • ID (use less; noun)
  • Class (adjective)

What is the difference between an ID and a Class?

ID can only be applied to one element and should be used less while Class can be applied to multiple elements. ID uses nouns while for class we use adjectives.

What is the difference between a SPAN tag and a DIV tag?

SPAN tag is for styling small parts of a text and it doesn't create a new line while the DIV tag is for creating sections and starts a new line.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV tag is for group of elements for styling on a page while the header is for the introductory section only
